# Léo Grognet — Game & Engine Programmer

I'm a game programming student with a focus on **low-level systems** and **gameplay architecture**.  
I like understanding how things work under the hood — and building them from scratch when needed.

---

## What I build

**Engine systems** — I contributed to [GCEP](https://github.com/LowLevelDevTeam/GCEP), a custom C++20 game engine built on Vulkan and Jolt Physics.  
My responsibilities: **ECS architecture** and **binary serialization system**.  
The engine includes a full editor (ImGui), a custom RHI layer over Vulkan, physics, audio, and CI/CD via GitHub Actions.

**Gameplay programming** — I work on player systems and game feel.  
On [ArenaShooter](https://github.com/ArenaShooter/ArenaShooter_2025/tree/PLAYER_BASE) (Unreal Engine 5, 1 week), I built the full player controller: jump, dash, slide, wall run, wall climb.  
On [Runner](https://github.com/leogrognet/Runner_Leo_Theo) (C++ / SFML, 2 weeks), I implemented a mini Model-Actor system from scratch to manage game entities.
